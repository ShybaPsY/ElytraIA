# Elytra IA - Sales AI Assistant Website

> Transform Conversations into Conversions

A modern, interactive website for Elytra IA - an AI-powered sales assistant that analyzes conversations, detects client emotions, and provides smart message suggestions to boost conversion rates from 10% to 35%.

## Features

### Design & UI
- **Polygonal Wings Logo** - Animated SVG logo with modern geometric design
- **Glassmorphism Effects** - Frosted glass design aesthetic throughout
- **Gradient Themes** - Beautiful purple/blue gradients with smooth transitions
- **Dark/Light Mode** - Toggle between themes with persistent preferences
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices

### Interactive Elements
- **Animated Hero Section** - Particle background that responds to mouse movement
- **Typing Effect** - Animated headline text
- **Scroll Animations** - Elements fade and slide in as you scroll
- **Counter Animations** - Statistics count up when visible
- **Interactive Demo** - Live chat simulation showing AI in action
- **FAQ Accordion** - Expandable/collapsible questions
- **Hover Effects** - Cards lift and glow on interaction
- **Ripple Buttons** - Material design ripple effect on clicks

### Sections
1. **Hero** - Compelling headline with animated wings logo
2. **Features** - 6 key AI capabilities with icons
3. **How It Works** - 4-step timeline with glassmorphic cards
4. **Live Demo** - Interactive chat showing emotion detection and AI suggestions
5. **Pricing** - 3 pricing tiers (Start, Team, Enterprise)
6. **Testimonials** - Customer reviews with ratings
7. **FAQ** - Common questions with accordion
8. **CTA** - Call-to-action with trial signup
9. **Footer** - Links, social media, legal info

### Technical Features
- **Smooth Scrolling** - Butter-smooth page navigation
- **Parallax Effects** - Background elements move at different speeds
- **Loading Screen** - Animated wings on page load
- **Floating Action Button** - Quick access to demo/contact
- **Scroll to Top** - Button appears when scrolling down
- **Performance Optimized** - Lazy loading and efficient animations
- **SEO Ready** - Semantic HTML structure

## Customization

### Colors
Edit CSS variables in `styles.css` (lines 8-25):
```css
:root {
    --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    /* Edit other colors here */
}
```

### Content
- **Pricing**: Edit plans in `index.html` (search for "pricing-card")
- **Features**: Modify feature cards (search for "feature-card")
- **Testimonials**: Update reviews (search for "testimonial-card")

### Logo
Replace the SVG code in the hero section to customize the wings design.
